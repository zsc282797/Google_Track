Sprint 1 Backlog.  
Mar 11 to Mar 18 


User Stories from Product Backlog that we will address in this sprint:
a) Being able to list files and sub directory of a directory in a normal way or recursive way
b) Being able to move a file from one directory to another
c) Being able to retrieve a file from a web address in url form 
d) Being able to recall and execute a command given an history number
e) Being able to find a certain word pattern [REGEX] and print the path to the file and the line which contains the pattern
f) Being able to display the contents of multiple files ignoring invalid paths
